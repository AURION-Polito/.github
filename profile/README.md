# AURION-Polito

AURION (Algorithms and Utilities for Research, Innovation, and Operations in Numerical analysis) is the official organization for the numerical analysis group at DISMA, Politecnico di Torino (Polito), Italy.

The official website is [polydim.it](https://www.polydim.it/).
